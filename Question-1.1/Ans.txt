--->> JavaScript is a scripting or programming language that allows you to implement complex features on web pages, It is a 
      high-  level, interpreted programming language that is often executed in web browsers to enhance the user experience by 
      making web pages interactive and responsive.

--->> Key aspects of JavaScript and its role in web development include:
  1.> Client-Side Scripting: JavaScript is mainly a client-side scripting language, meaning it runs in the user's browser 
      rather than on the server. This allows developers to create dynamic and interactive elements on the user's side, such as 
      form validation, animations.

  2.> Frameworks and Libraries: There are numerous JavaScript frameworks and libraries, such as React, Angular, and Vue.js, 
      that simplify and streamline the development process.
