--->> JavaScript operators are fundamental building blocks of the language, allowing developers to perform various operations on values. From basic arithmetic calculations to 
      complex logical evaluations, JavaScript operators provide the necessary tools to manipulate data and control program flow.

--->> There are the following types of operators in JavaScript.

     1.> Arithmetic Operators
     2.> Comparison (Relational) Operators
     3.> Bitwise Operators
     4.> Logical Operators
     5.> Assignment Operators
